# Validation

- Shell syntax: logged in shell-syntax.log.
- Build: run this script with --with-build, or inspect build.log if already present.
- Tests: run this script with --with-tests, or inspect test.log if already present.
- Runtime model validation: use start-low-gpu-summary.sh and inspect its summary.md plus run.log.
- Git verification: inspect git-status.log, git-history.log, and git-diff.log.
